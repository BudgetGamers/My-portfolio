How to Join

Follow these steps to join our community and servers

Step 1: Install Hamachi

The first step is to install hamachi

Click here to download Hamachi

Step 2: If you are on Linux

Skip to step 3 if you are on windows

You will need to install flatpak and haguichi

Click here to download Flatpak

Then install haguichi using flatpak

flatpak install flathub com.github.ztefn.haguichi

Step 3: setup hamachi

Click the power button in the top left of hamachi and it will promt you to sign in,
on windows it will ask for script acceptance press YES.
Click "Signup" and enter your email and password you wish to use with hamachi.

Step 4: Connect to the network

Connect to the BudgetGamers Network

Network: BudgetGamers[xx] Password: BudgetGamers

Replace [xx] with any number 01 up to 20.
(e.g. BudgetGamers01, BudgetGamers02, etc.)

Step 5: Connect to the server

Connect to the BudgetGamers Server of your choice

Click here to see the servers
