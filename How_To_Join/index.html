How to Join

Follow these steps to join our community and servers
If you are on Minecraft Bedrock Edition skip to Bedrock Step 1.

Step 1: Install Hamachi

The first step is to install Hamachi

Click here to download Hamachi for Windows Click here to download the installer for Linux

If you are on linux run the installer with sudo ./install.sh
and make sure you have it set as able to run as an executable in the propertys

Step 2: setup hamachi

Click the power button in the top left of hamachi and it will promt you to sign in,
on windows it will ask for script acceptance press YES.
Click "Signup" and enter your email and password you wish to use with hamachi.

Step 3: Connect to the network

Connect to the BudgetGamers Network

Network: BudgetGamers[xx]

Password: BudgetGamers

Replace [xx] with any number 01 up to 20.
(e.g. BudgetGamers01, BudgetGamers02, etc.)

Step 4: Connect to the server

Connect to the BudgetGamers Server of your choice

Click here to see the servers

Bedrock Step 1: Add BudgetGamers354 as a friend

Go to your friends list in Minecraft and add BudgetGamers354 as a friend

Bedrock Step 2: Join the crossplatform server

Join the crossplatform server by clicking on BudgetGamers354 in your friends list and clicking "Join Server"
