# Minimal Centered Portfolio

A premium, glassmorphic portfolio designed for GitHub Pages. 

## Features
- **Minimalist Centered Layout**: Linktree-style but with higher-end aesthetics.
- **Glassmorphism Design**: Frosted glass effects on a dark background blob system.
- **Social Integration**: Custom links for Discord, GitHub, CurseForge, and Google Slides.
- **Steam Friend Code**: Copy-to-clipboard functionality built-in.
- **Responsive**: Fully optimized for mobile and desktop screens.
- **Lucide Icons**: Clean, modern icons for all links.

## How to Deploy
1. **GitHub Pages**:
   - Push this repo to GitHub.
   - Go to **Settings > Pages**.
   - Select the `main` branch and `/root` or `/(root)` folder.
   - Save and your portfolio will be live at `https://yourusername.github.io/reponame`.

2. **Custom Domain**:
   - Update the `CNAME` file in the root if you have a custom domain.

## Customization
- Open `index.html` to update your name, bio, and social links.
- Open `style.css` to tweak colors or the background blob system.
- Replace the profile picture URL in `index.html` with your own image hosting link.
